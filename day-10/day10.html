1 HTML Introduction
  • It is a standard Markup Language.
  • It is used to create webpages.
  • It acts as structure to the website.
  • It contains series of elements which is represented by tags --->which tells the browser how to display the content.

    • Tags: An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document.
    Left Tag ...........< Reserved words >............Right Tag
    Types of Tags: There are two types of Tags 1. Container tags or Pair tags
    2. Empty tags or Void tags or Self closing tags.
    1. Pair Tags: Pair tags are tags which contains both starting tag and Ending tag.
    • Opening Tag: < tagname >
    • Closing Tag: < /tagname >
    2. Empty Tags: Empty tags are tags which doesnt required closing tag.
    EX: < tagname >

    Anatomy of Elements:

    Elements which consists of opening tag, content and closing tag.
    Structure of Element: “< opening tag >Content< closing tag >”
    Series of Elements: It is nothing but of both opening tag and closing tag are same.
    Ex:< tag1 > Fincorp< /tag1 > < tag2 > about < /tag2 > Nested Elements: It is nothing but of one element can be placed in another element is known as Nested elements. Ex: < parent tag >
    < child tag >
    < grandchild > < /grandchild >
    < /child tag >
    < /parent tag >
    HTML: html is a root element for the html document < html > some content < /html >.
    2 HTML Structure < !DOCTYPE>: It tells the version of document
    < html >< /html >: It is a root element.
    < head >< /head >: It is head tag contains title and meta information.
    < body >< /body >: It contains content related of the website.

    4 Comments and Table Attributes Comments: It used for documentation of code, not visible in browser and prevents the execution of the code.

    Table: It is used to add the Information in a tabular format.

  • Tables have rows and coloms.
  • tables are defined with table tag.
  • Table rows are defined with "tr" tag.
  • Table cells are defined with "td" tag.
  • Heading tags are defines with "th" tag.
    • Ex:< table >
    < tr >
    < td >employe name< /td >
    < /tr >
    < /table >

    Attributes of Table:
    1. border: It is used to specify border for a table.
    2. caption: It is used to provide title or heading.
    Ex: < caption >< h2 >Student Table< /h2 >< /caption >
    3. cellpadding: It is used to create a space around the content.
    Ex: < table border="2" cellpadding="20" > Content < /table >
    4. cellspacing>: It is used to create a space between the cells.
    Ex: < table border="2" cellpadding="20" cellspacing="10" > Content < /table >
    5. Grouping: Grouping the cells can be done with thead, tbody, tfoot and also used for styling.
    Ex: < table >
    < thead > < tr >
    < th >employe name< /th >
    < /tr >
    < /thead >
    < /table >
    6. bgcolor: It is used to create a styling background color.
    Ex: < table bgcolor="Green" border="2" cellpadding="20" cellspacing="10" > Content < /table >
    7. Sizing: It contains width and hight of the table.
    Ex: < table border="2" cellpadding="20" cellspacing="10" width="50" or "50%" hight="50" or "50%" > Content < /table >
