EMMETS

Emmets are the shortcut abbrivations for writing the html and css code

EMMETS TABLE
Emmets Description
! (or) html:5 Open html Structure
Tagname By using tagname
> Child format
Ex: table>tr>td
+ Siblings format
Ex: table>caption+tr
* Multiple format
Ex: table>tr>td*5
{} Content
Ex: table>tr>td{content}*3
^ Climb up
Ex: html>head>title^body
[] Attributes
Ex: attribute name= "value"
$ Naming and numbering
Ex: h$[title=topic$]{Headline $}*3
() Grouping
Ex: table>(tr>td>*2>a)
